Сервис GitHub

1) Зарегистрируйтесь на сервисе GitHub (если вы не зарегистрированы)

2) Создайте репозиторий

3) Озаглавьте его

4) Нажмите на Create repository

5) Посмотрите это видео

6) Теперь, когда вы освоили этот язык, напишите что-нибудь на Html редакторе, можете выбрать другой редактор, который вам понравится.

7) Создайте текстовывй документ, вставьте туда свой код, а потом сохраните этот текстовый документ как "index.html". Вам высветет сообщение что файл может оказаться недоступным, не волнуйтесь, в любой момент можно будет переименовать файл обратно в txt.

9) Проверьте правильность сайта нажав на получившийся файл

10) Если все правильно, то перекиньте файл по этой кнопке:

11) После, нажмите на Commit Changes

12) Зайдите в settings > pages, и поменяйте Branch как на картинке:

13) Нажмите на "Save" и вставьте эту ссылку в браузер "http://username.github.io/repository" (!НО ВМЕСТО username ВВЕДИТЕ СВОЕ ИМЯ В GitHub, А ВМЕСТО repository ВВЕДИТЕ ИМЯ КОТОРОЕ ВЫ ВПИСЫВАЛИ ДЛЯ СВОЕГО РЕПОЗИТОРИЯ!)
